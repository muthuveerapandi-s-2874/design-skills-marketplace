# Zoho Theme Factory

Apply consistent, on-brand Zoho ODS styling to any artifact: HTML pages, PPTX decks, DOCX documents, and social media banners.

All 15 themes — Brand and Creative — use **only ODS color tokens** and **Zoho Puvi** typography. No arbitrary colors are ever introduced.

---

## Usage Workflow

1. **Identify the artifact type** — HTML, PPTX, DOCX, or social media banner
2. **Present both theme groups** — show the 5 Brand themes and 10 Creative themes
3. **Ask the user to choose** — or match their described vibe to the closest theme
4. **If no theme fits or user is unsure** → apply **ODS Light** (default fallback)
5. **Apply the theme** using the token specs below

## Artifact-Specific Application Notes

### HTML Landing Pages
- Set `font-family: 'Zoho Puvi', sans-serif` on `body`
- Background color on `body`; surface color on `.card` / `.section-alt`
- Heading color on all `h1–h6`; body text color on `p`
- Card elevation: `box-shadow: 0px 16px 24px -4px rgba(0,0,0,0.25)`
- Grid: `max-width: 1400px; margin-inline: auto; padding-inline: 40px`

### PPTX Slide Decks
- Slide background = theme background color
- Title: Zoho Puvi Bold (700), heading color
- Body: Zoho Puvi Regular (400), body text color
- Accent shapes / lines: primary accent color
- Cover slide: primary accent as full background, white text

### DOCX Word Documents
- Document theme color = primary accent
- Heading 1: Bold 700, heading color, 32pt
- Heading 2: Semibold 600, heading color, 24pt
- Body: Regular 400, body text color, 11pt
- Table headers: surface color background
- Hyperlinks: primary accent, no underline

### Social Media Banners
| Format | Dimensions |
|---|---|
| LinkedIn post | 1200 × 627px |
| Twitter/X post | 1600 × 900px |
| Instagram square | 1080 × 1080px |
| Instagram story | 1080 × 1920px |
| Facebook cover | 820 × 312px |

Use theme background for the canvas, heading color for headline text, primary accent for geometric shapes/borders and icon fills.

## Quality Checklist

- [ ] Zoho Puvi loaded / specified
- [ ] All hex values sourced from this skill — no arbitrary colors
- [ ] Border-radius values only: 2, 4, 8, 16, or 160px
- [ ] Text contrast passes WCAG AA
- [ ] CTA buttons use `border-radius: 4px`
- [ ] Layout uses `40px` grid margin
- [ ] Heading letter-spacing applied
- [ ] Theme is internally consistent — no color mixing across themes
